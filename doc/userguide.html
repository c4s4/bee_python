This file is the reference documentation for the Python Bee package. This package is made of tasks to build your Python projects and templates to quick start new ones.

Installation

Bee packages are managed as Ruby Gems, thus you can install Bee Python typing sudo gem install bee_python under Unix systems or gem install bee_python under Windows. Note that this will also install Bee if not already installed because Bee is a required dependency of bee_python.

Python Tasks

To get a reference about Python tasks, please see Tasks Reference section.

You can also get documentation about a given task in a terminal typing bee -k python.pylint for instance. To list all available tasks in the Python package, type bee -k python.?.

Python Templates

To get a reference about Python templates, please see Templates Reference section.

You can also get documentation about a given template in a terminal typing bee -e python.project for instance. To list all available templates in the Python package, type bee -e python.?.

Enjoy!
